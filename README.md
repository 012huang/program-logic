# program-logic
微信公众号"老马说编程"中的代码示例，所有示例代码基于JDK 7。

标准Eclipse项目，导入即可，导入后可能需要修改JDK配置。

包结构：

* 基础：shuo.laoma.basic
* 类：shuo.laoma.cls
* 常见类：shuo.laoma.commoncls
* 泛型和容器类：shuo.laoma.collection
* 文件：shuo.laoma.file
* 并发：shuo.laoma.concurrent
